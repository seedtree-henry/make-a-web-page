Webpages, Documents, and Structure

1. World Wide Web

Web shows any contents using HTML. Web shows not only texts but also images and videos. Also, web connects web pages using hyperlinks.

Compoents of web are described below.
- HTTP : The main protocol of the web
- Servers : Computers that host the files that make up the web
- Internet : The world's largest computer network
- Brower : A program that runs on your computer to display files found on the web

2. HTML

HTML stands for Hypertect Markup Language. HTML is composed of text contents, markup, references to other documents and links to other pages.
This video explains what is HTML briefly.

3. Markup

Markup tells what it looks like on the web.

4. Inline / Block

- Inline : Shows texts inline.
- Block : Elements are in an invisible box.

5. HTML, CSS and Javascript

- HTML : The structure of a webpage
- CSS : The style of the webpage
- Java script : Interactive components

Note) All elements are rectangular. Even though the shape is a circle, it is a circle in an invisible rectangle.
By using invisible retangles, you can manage elements well because repositioning elements is easy.
